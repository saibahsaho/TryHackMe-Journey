# Linux Fundamentals

## Commands I used 
nano - create and edit files
cat - display file contents
find - search for files and directories
grep - search for text inside files
pwd - show current directory
ls - list files and folders 

## Reflection 
At first, using Nano and navigating  Linux directories felt confusing, but after practicing the commands directly in the terminal, I became more comfortable using the Linux environment.
This part helped me understand how important terminal-based file editing is in cybersecurity and system administration.

## What I learned
Linux text editors like Nano and Vim are essential tools used by cybersecurity professionals, developers and system administrators.
I also learned that Linux command-line practice improves troubleshooting, navigation and file management skills.
