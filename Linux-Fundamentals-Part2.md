# Linux Fundamentals part 2

## What I covered 
File permissions, flags and operators 

## Commands I used 
ls -l - view file permissions
cat - read files
find - search for files 
grep - search inside files
touch - create files
mkdir - create directories
mv - move/rename files 
rm - remove files
cp - copy files


## Reflection
This part helped me understand how important file navigation and management are in Linux. The commands started feeling more natural after practicing them multiple times.

## What I learned 
Linux file management is essential in cybersecurity because tools, scripts, logs and configurations are all handled through the terminal.
